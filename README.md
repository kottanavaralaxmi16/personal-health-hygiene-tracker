# Personal Health & Hygiene Tracker

## Description
A Python-based application to track daily health and hygiene habits such as water intake, sleep, exercise, and handwashing.

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Run `health_tracker.py` to input daily data
2. Run `analysis.py` for analysis and suggestions
3. Run `report.py` to visualize trends

## Outcome
Helps users identify unhealthy habits and improve lifestyle practices.
